Ultra-fast Search Library
Apache Lucene set the standard for search and indexing performance. Lucene is the search core of both Apache SolrTM and ElasticsearchTM.
Proven search capabilities
Our core algorithms along with the Solr search server power applications the world over, ranging from mobile devices to sites like Twitter, Apple and Wikipedia.
Large, Vibrant community
